Shell permissions project
what do the commands chmod, sudo, chown, chgrp do.
0-iam_betty - su switches current user to the new user betty.
1-who_am_i - prints username of the current user.
2-groups - prints all the groups of the current user.
3-new_owner - writes script that changes owner of file to new user.
4-empty - creates empty file called hello.
5-execute - adds execute permission to the owner of the file hello.
6-multiple_permissions - executes permission to the owner and the group owner and read permission to the other users to the file hello.
7-everybody - adds execution permission to the owner group owner and other users to the file hello.
8-James_Bond - executes 007 permissions to hello.
9-John_Doe - script that sets mode to -rwxr-x-wx.
10-mirror_permissions - script that changes file permissions to the same as olleh.
11-directories_permissions - script that executes permission to all suddirectories.
12-directory_permissions - script that creates directory with permissions 751 in working directory.
