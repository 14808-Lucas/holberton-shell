Shell permissions project
what do the commands chmod, sudo, chown, chgrp do.
0-iam_betty - su switches current user to the new user betty.
1-who_am_i - prints username of the current user.
2-groups - prints all the groups of the current user.
3-new_owner - writes script that changes owner of file to new user.
4-empty - creates empty file called hello.
5-execute - adds execute permission to the owner of the file hello.
